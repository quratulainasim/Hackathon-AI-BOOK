# Summary Maker Skill

## Description
This skill helps create concise, well-structured summaries of text content, documents, or code.

## Capabilities
- Generate summaries of various lengths (brief, medium, detailed)
- Identify key points and main ideas
- Maintain important context while reducing length
- Format summaries for readability

## Usage Guidelines
- Provide the text or content that needs summarizing
- Specify desired summary length or number of key points
- Indicate if technical terms should be preserved
- Mention if the summary should be in bullet points or paragraph form

## Best Practices
- Always preserve the core meaning and intent of the original content
- Focus on the most important information first
- Use clear, concise language
- Maintain the tone appropriate to the original content
