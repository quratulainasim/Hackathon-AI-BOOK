# Translation Helper Skill

## Description
This skill assists with translating text between languages while maintaining context, tone, and cultural appropriateness.

## Capabilities
- Translate text between multiple languages
- Preserve meaning and context during translation
- Adapt idioms and expressions appropriately
- Maintain formal/informal tone as needed
- Handle technical terminology correctly

## Supported Languages
- English
- Spanish
- French
- German
- Chinese (Simplified and Traditional)
- Japanese
- Korean
- Portuguese
- And many other languages

## Usage Guidelines
- Specify source and target languages
- Provide context about the text type (technical, casual, formal, etc.)
- Indicate if certain terms should remain untranslated
- Mention if cultural adaptation is needed beyond literal translation

## Best Practices
- Preserve the original intent and meaning
- Consider cultural nuances in the target language
- Maintain the appropriate register (formal vs informal)
- Verify technical terms for accuracy
- Flag ambiguous phrases that may need human review
